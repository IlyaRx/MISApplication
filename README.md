# MISApplication

MISApplication — это приложение для записи людей на приём к врачу. Проект разработан на C# с использованием WPF и базы данных  MSSQL через Entity Framework.

## Функциональность

*   **Управление данными:** Добавление, просмотр, редактирование и удаление записей приёма у врача к каждому пациенту.
*   **Поиск и фильтрация:** Быстрый поиск по базе данных.
*   **Авторизация:** Разграничение прав доступа для разных пользователей.

## Стек технологий

*   **Язык:** C#
*   **Платформа:** .NET Framework / .NET Core 5
*   **Интерфейс:** WPF
*   **База данных:** Entity Framework, MSSQL SQL Server
*   **Инструменты:** Git, Visual Studio

## Установка и запуск

### Требования

*   [.NET Framework / .NET SDK](https://dotnet.microsoft.com/download)
*   SQL Server
*   Visual Studio 2019/2022

### Инструкция по запуску

1.  **Клонировать репозиторий**
    ```bash
    git clone https://github.com/IlyaRx/MISApplication.git
    cd MISApplication
    ```
2. **Открыть решение**
    Открой файл MISApplication.sln в Visual Studio.

3. **Восстановить пакеты NuGet**
    В Visual Studio: правой кнопкой по решению -> "Восстановить пакеты NuGet".

4. **Настроить базу данных**
    Проверь строку подключения в файле App.config или appsettings.json.

5. Запустить приложение
    Нажми F5 в Visual Studio.
